Sobre o Projeto

👨‍🎓 Este projeto foi criado como parte da resolução do desafio proposto pelo Programa One Next Education, uma parceria da Oracle com a Alura. A API se conecta a uma base de dados de livros e autores, permitindo operações diversas para explorar e persistir informações.

🔨 Etapas do Desenvolvimento

  • ⚙️ Configurar o ambiente Java e Spring
  • 🌐 Conhecer e interagir com a API
  • 📡 Construir solicitações e analisar JSON
  • 🔄 Converter e persistir dados
  • 📚 Consultar livros e autores
  • 🔎 Listar autores vivos em ano específico
  • 🌍 Listar livros por idiomas

💻 Ferramentas e Tecnologias Utilizadas

  • Java com Spring Boot
  • IntelliJ IDEA como ambiente de desenvolvimento
  • PostgreSQL para persistência
  • Consumo e manipulação de JSON

Competências: API REST · Java · Spring Boot · PostgreSQL · Análise de Dados · Programação Orientada a Objetos
