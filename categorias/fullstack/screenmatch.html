Sobre o Projeto

📺 Este projeto foi desenvolvido durante o segundo curso da formação Avançando com Java da Alura. A aplicação, evolução do projeto Screenmatch, utiliza Spring Data JPA para persistência e realiza consultas em um banco PostgreSQL, consumindo também dados da API do ChatGPT.

🔨 Etapas do Desenvolvimento

  • 🧩 Criar menu com múltiplas opções e funcionalidades
  • 📦 Modelar abstrações com classes, enums, atributos e métodos
  • 🤖 Integrar com API do ChatGPT
  • 🗃️ Utilizar Spring Data JPA para persistência no banco
  • 🔍 Explorar vários tipos de consultas com JPARepository
  • 🐘 Utilizar PostgreSQL como banco relacional

💻 Ferramentas e Tecnologias Utilizadas

  • Java como linguagem principal
  • Spring Boot + Spring Data JPA
  • PostgreSQL como banco de dados
  • API do ChatGPT para enriquecimento de dados
  • Git e GitHub para versionamento

Competências: Java · Spring Boot · JPA · PostgreSQL · APIs externas · Modelagem de Domínio · Persistência de Dados · Consultas Avançadas · Versionamento com Git
