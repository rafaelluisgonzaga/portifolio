Cartão de Crédito

Aplicação Java para controle de transações e limite de crédito

Ver no GitHub

Sobre o Projeto

💳 Este projeto tem como objetivo implementar uma aplicação Java que permita registrar compras realizadas em um cartão de crédito. Com foco em orientação a objetos, o sistema foi estruturado com classes específicas para representar as transações e o controle de limite do cartão.

🔨 Etapas do Desenvolvimento

  • 🧾 Classe Compra: representa cada transação realizada
  • 💰 Classe CartaoCredito: controla limite disponível e histórico de compras
  • 🧠 Lógica principal: implementada no método main com interação via terminal

💻 Ferramentas e Tecnologias Utilizadas

  • IDE: IntelliJ IDEA CE
  • Linguagem: Java
  • Paradigma: Programação Orientada a Objetos (OOP)
  • Interface: Terminal
  • Versionamento: Git & GitHub

Competências: Programação em Java · Orientação a Objetos · Controle de Limites · Entrada de Dados via Terminal · Git · GitHub · Modelagem de Classes
