Sobre o Projeto

🎧 Projeto desenvolvido durante a Imersão Front-End 2ª Edição da Alura, com foco na construção de uma página estilizada inspirada na interface do Spotify. A aplicação utiliza técnicas modernas de HTML, CSS e JavaScript, com foco em responsividade e boas práticas de desenvolvimento front-end.

🔨 Etapas do Desenvolvimento

  • Aula 1: Revisão de HTML e CSS
  • Aula 2: Estilo avançado e posicionamento
  • Aula 3: Layout com Flexbox, pseudo-classes e responsividade
  • Aulas 4 e 5: Conteúdo em desenvolvimento

💻 Ferramentas e Tecnologias Utilizadas

  • Linguagem: JavaScript
  • Editor: VS Code
  • Versionamento: Git & GitHub

Competências: HTML · CSS · JavaScript · Flexbox · Pseudo-classes · Responsividade · Git · GitHub · Desenvolvimento Web
