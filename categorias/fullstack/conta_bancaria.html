Sobre o Projeto

🧑‍💼 Este projeto foi desenvolvido como parte da resolução de desafio proposto pelo Programa One Next Education, uma parceria entre Oracle e Alura. A aplicação permite gerenciar dados de clientes e realizar operações básicas como visualização de saldo e movimentações financeiras — tudo via terminal, com estrutura orientada a objetos.

🔨 Etapas do Desenvolvimento

  • 📄 Inicialização dos dados do cliente
  • 📋 Menu de opções no terminal
  • 👁️ Visualização do saldo
  • 💸 Envio de valor
  • 💰 Recebimento de valor

💻 Ferramentas e Tecnologias Utilizadas

  • IntelliJ IDEA como ambiente de desenvolvimento
  • Java JDK 17 como plataforma principal
  • Luri a IA da Alura

Competências: Programação em Java · Terminal · Estrutura de Menus · Lógica de Negócio · Entrada e Saída de Dados · Versionamento com Git · Gestão de Transações
