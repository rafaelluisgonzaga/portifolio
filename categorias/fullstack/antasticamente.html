Sobre o Projeto

🐜 Este projeto foi desenvolvido durante a Imersão Back-End da Alura com foco na construção de uma aplicação moderna que apresenta dados sobre espécies de formigas. A interface conecta informações geográficas e imagens por meio de integração com APIs externas, com estrutura robusta utilizando Node.js, Express e MongoDB.

🔨 Etapas do Desenvolvimento

  • 📄 Exibição de espécies de formigas
  • 🖼️ Imagens com descrição gerada por IA (Google Gemini)
  • 🌍 Dados geográficos integrados via API
  • 🗃️ Armazenamento e busca de dados no MongoDB

💻 Ferramentas e Tecnologias Utilizadas

  • HTML, CSS e JavaScript: interface visual atrativa
  • Node.js e Express: backend robusto
  • MongoDB: banco de dados NoSQL
  • APIs externas: informações geográficas e imagens
  • Google Cloud: infraestrutura de deploy

Competências: Node.js · Express.js · MongoDB · APIs REST · JavaScript · HTML/CSS · Web Deployment · Google Cloud · Backend Moderno · Full Stack
