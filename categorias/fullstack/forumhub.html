Sobre o Projeto

🚀 Apresento o ForumHub API — uma aplicação backend desenvolvida em Java 17 com Spring Boot, focada em gerenciamento de tópicos de fórum. Permite que usuários autenticados criem, editem, visualizem e excluam postagens, fomentando discussões seguras e organizadas.

🔨 Etapas do Desenvolvimento

  • 📝 Criação e Edição de Tópicos
  • 👀 Visualização Detalhada
  • ❌ Exclusão Controlada
  • 🔐 Autenticação com Spring Security

💻 Ferramentas e Tecnologias Utilizadas

  • Spring Boot, Spring Data JPA
  • Flyway para migração de banco de dados
  • MySQL + Hibernate
  • Swagger para documentação futura

Competências: API REST · Java · Spring Boot · Segurança · Banco de Dados · Versionamento com Git
