Instagram Post Builder

Sistema com agentes Google para criação automatizada de conteúdo

Ver no GitHub

Sobre o Projeto

🤖 Este projeto implementa um sistema automatizado de criação de posts para Instagram utilizando agentes em cadeia via Google ADK. O sistema é capaz de buscar notícias recentes, planejar o conteúdo, redigir e revisar um post com base em tendências sobre um tópico fornecido. Tudo isso de forma modular e extensível. 🧠📲

🔨 Etapas do Desenvolvimento

  • 🗞️ Agente buscador de notícias
  • 🧠 Planejador de conteúdo do post
  • ✍️ Redator do rascunho para Instagram
  • ✅ Revisor de qualidade final

💻 Ferramentas e Tecnologias Utilizadas

  • 🧠 Google ADK + Gemini
  • 🐍 Python 3 + IPython
  • 🌐 Google Search API
  • 📦 requests · datetime · google-genai

Configuração: Executado em ambiente Jupyter ou Google Colab com chave de API Google. Sistema pronto para escalar e adaptar para múltiplas redes sociais.
