Sobre o Projeto

💬 Este projeto demonstra como criar um chatbot simples utilizando a API Google Gemini dentro do ambiente Google Colab. Desenvolvido durante o evento Imersão IA Gemini 3 da Alura em 2025, o notebook fornece uma interface de linha de comando para interagir com modelos de linguagem, mantendo histórico de conversa e possibilitando configuração de sessões inteligentes. 🚀🧠

🔨 Etapas do Desenvolvimento

  • ⚙️ Instalação da biblioteca google-genai
  • 🔑 Autenticação com chave via userdata
  • 📄 Listagem e escolha de modelos Gemini
  • 🧠 Configuração e manutenção de sessões de chat
  • 💬 Interface interativa de linha de comando

💻 Ferramentas e Tecnologias Utilizadas

  • 🔗 Google Gemini API
  • 🐍 Python 3 + Google Colab
  • 📦 google-genai
  • 📜 Licença MIT

Configuração: Executado em ambiente Google Colab, utilizando chave de API armazenada nos Segredos do Colab. Requer conta Google para integração.
