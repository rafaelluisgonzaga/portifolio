Sobre o Projeto

🎙️ Este projeto foi desenvolvido ao vivo no YouTube em parceria com a DIO, com o objetivo de automatizar a produção de um podcast usando ferramentas de Inteligência Artificial. A proposta envolve o uso de uma esteira de prompts para gerar roteiro, voz, capa e edição final. 🔊🤖

🔨 Etapas do Desenvolvimento

  • ✏️ Criar roteiro com prompts no ChatGPT
  • 🔈 Gerar áudio com os roteiros no ElevenLabs
  • 🖼️ Criar artes com prompts no DALL·E 3
  • 🎚️ Tratar áudio e montar episódio com CapCut

💻 Ferramentas e Tecnologias Utilizadas

  • 🧠 ChatGPT para criação de roteiros
  • 🎙️ ElevenLabs para geração de voz
  • 🖼️ DALL·E 3 para capas visuais
  • 🎧 CapCut para edição de áudio e trilha sonora

Materiais: Template Notion · Editor de áudio · Sequência de prompts otimizados para cada etapa
