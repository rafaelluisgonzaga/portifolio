Sobre o Projeto

🌐 Este repositório apresenta uma solução simulada de tradução de textos com Microsoft Azure Translator, desenvolvida como parte da certificação AI-102 da DIO. Utilizando Python com Jupyter Notebook, o projeto simula chamadas à API, permitindo aprendizado prático sobre provisionamento e integração de serviços Azure. 📘🚀

🔨 Etapas do Desenvolvimento

  • 🔧 Provisionamento de Azure Translator e Azure OpenAI
  • 🧠 Simulação via Python com scraping e tradução
  • 📘 Documentação completa em PDF (PT e EN)
  • 🔗 Pronto para conexão real com API e produção

💻 Ferramentas e Tecnologias Utilizadas

  • 🌐 Microsoft Azure Translator (simulado)
  • 🤖 Azure OpenAI + LangChain (simulado)
  • 🐍 Python 3 + Jupyter Notebook
  • 📄 BeautifulSoup4 para extração de conteúdo

Formação Relacionada: AI-102: Designing and Implementing an Azure AI Solution · Conclusão: Maio de 2025 — pela DIO
