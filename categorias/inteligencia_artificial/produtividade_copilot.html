Sobre o Projeto

💼 Este projeto compila os aprendizados do Boot Camp Python Developer da DIO em parceria com Suzano, focando na utilização do Microsoft Copilot Studio para criação de assistentes virtuais inteligentes. A iniciativa explorou desde configuração de ambiente até a criação de Copilots personalizados. 🧠⚙️

🔨 Etapas do Desenvolvimento

  • 🧾 Conta Microsoft 365 E3 Trial
  • 📲 Acesso via Power Platform
  • 🛠️ Copilot Studio configurado no ambiente de desenvolvimento

💻 Ferramentas e Tecnologias Utilizadas

  • 📌 Copilot de modelo ("Safe Travels")
  • 🤖 Copilot baseado em IA para respostas contextualizadas
  • ⚙️ Copilot criado do zero para soluções personalizadas

Caso Prático: Assistente virtual para viagens internacionais partindo do Brasil. Responde dúvidas e fornece requisitos personalizados de embarque.

Instrutor: Renato Romão de Souza · Microsoft MVP · MCT · Especialista em Copilot & IA
