Ver no GitHub

Sobre o Projeto

📘 Este projeto demonstra como utilizar ferramentas de IA para gerar um ebook completo, desde a criação do conteúdo textual até o design visual. O material foi construído ao longo de aulas ministradas como instrutor técnico na plataforma DIO, com foco em boas práticas, estética e autoridade técnica no nicho de CSS. 🧠📚

🔨 Etapas do Desenvolvimento

  • Título: 5 variações épicas com tema “Lanterna Verde” para CSS
  • Conteúdo: Lista de seletores CSS com explicações claras e exemplos reais
  • Imagem: “Green Lantern in spaceship with books and notebooks” no DALL·E 3

💻 Ferramentas e Tecnologias Utilizadas

  • 🧠 ChatGPT para conteúdo e estrutura textual
  • 🎨 DALL·E 3 para capas ilustradas temáticas
  • 📐 Keynote para diagramação e layout visual

Instruções: Utilize os prompts nas ferramentas indicadas, gere o conteúdo base e finalize a diagramação com editores como PowerPoint, LibreOffice ou InDesign.
