Sobre o Projeto

📝 Este projeto tem como foco a geração automatizada de documentos PDF contendo análises personalizadas. Utilizando agentes com modelos de linguagem como o Gemini, é possível produzir relatórios, diagnósticos e comunicações adaptadas a diferentes públicos. A automação ocorre via Python, com integração em notebooks e possibilidade de exportação direta em PDF. 📄⚙️

🔨 Etapas do Desenvolvimento

  • 💡 Análise de sentimentos com Gemini
  • 📊 Detecção de entidades para relatórios
  • 🧾 Geração de textos personalizados
  • 📥 Exportação final em PDF via Notebook

💻 Ferramentas e Tecnologias Utilizadas

  • 📘 Google Gemini + LangChain
  • 📍 Python 3 + IPython Notebooks
  • 🔍 PDFKit / FPDF para geração de documentos
  • 📂 Projeto modular e versionado no GitHub

Documentos Gerados: Resumos técnicos, análises políticas de países, diagnóstico de comunicação personalizada e cenário global 2024.
