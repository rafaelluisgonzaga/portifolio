Sobre o Projeto

🍷 Este case tem como foco a aplicação de técnicas de análise de dados e visualização para identificar padrões de consumo no mercado europeu de vinhos. Desenvolvido dentro do programa Santander Tech+ – Data Science by Ada Tech, oferece uma abordagem prática com dados simulados para apoiar decisões estratégicas. 📊📈

🔨 Etapas do Desenvolvimento

  • 📓 notebooks/: análises e visualizações em Jupyter
  • 📊 data/: conjunto de dados simulados
  • 📈 visualizacoes/: gráficos e dashboards gerados
  • 📘 README.md: orientações e resumo do projeto

💻 Ferramentas e Tecnologias Utilizadas

  • 🐍 Python · Jupyter Notebook
  • 🧮 Pandas · Matplotlib · Seaborn · Plotly
  • 📊 Visualizações interativas e mapas de calor

Principais Insights: O vinho tinto demonstrou maior aceitação; acidez volátil tem correlação negativa com qualidade; variáveis físico-químicas indicam padrões de avaliação.

Visualizações: Gráficos de dispersão, mapas de correlação, histogramas e boxplots por tipo e qualidade.
