Sobre o Projeto

💳 Este projeto tem como objetivo implementar um sistema bancário simples utilizando apenas funcionalidades nativas da linguagem Python. O usuário pode realizar depósitos, saques com regras de limite diário e consultar um extrato detalhado das movimentações. O exercício foi proposto para consolidar conceitos como variáveis, estruturas condicionais e loops. 🧠🐍

🔨 Etapas do Desenvolvimento

  • 💰 Implementação da função de depósito com validação
  • 🏦 Configuração de limite diário e valor máximo por saque
  • 📋 Registro de todas as operações e cálculo do saldo
  • 🧾 Exibição formatada do extrato final

💻 Ferramentas e Tecnologias Utilizadas

  • 🐍 Linguagem: Python 3.x
  • 📝 Editor: VSCode · PyCharm · Terminal
  • 📦 Nenhuma biblioteca externa utilizada
  • 📁 Execução via linha de comando com menus interativos
