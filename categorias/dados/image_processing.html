Sobre o Projeto

📦 O Image processing é um pacote Python desenvolvido para facilitar o processamento e visualização de imagens. Ideal para projetos que exigem análise de similaridade, equalização de histogramas ou redimensionamento automatizado. Também inclui utilitários para salvar, visualizar e comparar imagens com praticidade. 📊🧠

🔨 Etapas do Desenvolvimento

  • 🧱 Definição das funções essenciais para processamento de imagem
  • ⚙️ Implementação dos módulos `processing` e `utils`
  • 🧪 Testes com imagens reais para validação
  • 📦 Empacotamento e publicação para instalação via pip

💻 Ferramentas e Tecnologias Utilizadas

  • 🐍 Python 3
  • 🔬 scikit-image · NumPy · matplotlib
  • 📦 setuptools · PyPI · twine
  • 📁 Estrutura modular com documentação no README

Instalação: pip install image-processing-rafaelluisgonzaga
