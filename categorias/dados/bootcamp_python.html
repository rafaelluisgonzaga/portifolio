Sobre o Repositório

🐍 Este repositório reúne os exercícios práticos desenvolvidos durante o Bootcamp Suzano – Python Developer promovido pela DIO. Os arquivos incluem exemplos de sintaxe básica, operadores aritméticos, estruturas condicionais e fundamentos da linguagem Python. Ideal para quem está começando na programação e deseja consolidar os conceitos iniciais com código limpo e funcional.

📁 Estrutura do Repositório

  • 📄 README.md: informações gerais do projeto
  • 📂 Intro/: primeiros passos com Python
  • 📂 operadores_aritimeticos.py: exemplos de operações matemáticas

💻 Tecnologias Utilizadas

  • 🐍 Python 3.x
  • 🖥️ Editor: VSCode · PyCharm · Terminal
  • 📦 Sem bibliotecas externas
  • 📚 Foco em sintaxe, lógica e boas práticas

Bootcamp: Programa gratuito da DIO em parceria com a Suzano, voltado para iniciantes em programação com foco em Python e integração com IA.
