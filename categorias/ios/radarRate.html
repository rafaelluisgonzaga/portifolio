Sobre o Projeto

Durante o Santander Bootcamp 2023 - Mobile iOS na DIO, desenvolvi um app de câmbio com orientação do professor Robson. Esse projeto foi essencial para meu aprendizado em arquitetura MVVM e programação reativa com Combine, usando SwiftUI para criar interfaces fluidas e modernas.

💡 Funcionalidades Principais

  • ✅ Implementação de MVVM
  • ✅ Interface dinâmica com SwiftUI
  • ✅ Gerenciamento assíncrono com Combine
  • ✅ Conexão com API de câmbio
  • ✅ Busca e filtragem de dados

📲 Tecnologias Utilizadas

  • Swift · MVVM · Combine
  • API de câmbio com atualização em tempo real
  • Arquitetura reativa com separação clara de responsabilidades

Competências: Mobile Applications · MVVM (Model-view-viewmodel) · Programação (computação) · Combine
