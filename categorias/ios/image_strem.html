ImageStream App

Exploração de imagens com SwiftUI, Combine e API Pexels

Ver no GitHub

Sobre o Projeto

🧾 ImageStream é um aplicativo desenvolvido com SwiftUI que consome imagens da API do Pexels. Utiliza Combine para reatividade e gerenciamento de estado via ObservableObject. Criado como parte do Santander Bootcamp 2023, o app exibe fotos aleatórias de forma dinâmica e elegante.

💡 Funcionalidades

  • 🖼️ Seleção e exibição de imagens aleatórias da API Pexels
  • 🔄 Atualização automática com Combine
  • 🧠 Arquitetura MVVM
  • 📱 Interface responsiva e adaptada para dispositivos móveis

🛠️ Tecnologias Utilizadas

  • SwiftUI com ObservableObject
  • Combine para fluxo reativo
  • Swift Package Manager para gerenciamento de dependências
  • API REST fornecida pelo Pexels

Competências: Swift · Combine · SwiftUI · MVVM · API Integration · iOS Development
