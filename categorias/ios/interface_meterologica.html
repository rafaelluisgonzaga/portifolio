Sobre o Projeto

☀️ Interface Meteorológica é um aplicativo desenvolvido com SwiftUI para exibir previsões do tempo de forma visual e moderna. O projeto utiliza arquitetura MVVM, integração com API climática, e uma interface responsiva com ícones e cores dinâmicas.

💡 Funcionalidades

  • 🌡️ Conversor de temperatura
  • 🗓️ Previsão semanal
  • 🎨 Sistema de cores com Assets.xcassets
  • 📱 Design responsivo e adaptado para iPhone

🛠️ Tecnologias

  • SwiftUI com MVVM
  • Uso de ViewModel para lógica de exibição
  • Imagens e ícones otimizados com Assets
  • Xcode e GitHub para versionamento

Competências: Swift · SwiftUI · MVVM · APIs · Design Responsivo · iOS Architecture
