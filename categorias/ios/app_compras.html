Sobre o Projeto

🛒 App Compras é um app direto ao ponto: adicionar e remover itens da sua lista de compras. Desenvolvido com UIKit e Auto Layout, ele utiliza componentes básicos como UITableView, UserDefaults e protocolo Delegates para garantir uma navegação fluida e dados persistentes. Ideal para quem está iniciando no universo iOS!

📋 Funcionalidades

  • ➕ Adição de novos itens com validação
  • 🗑️ Exclusão de itens com swipe na UITableView
  • 📱 Interface responsiva com Auto Layout
  • 💾 Persistência usando UserDefaults

🔧 Tecnologias

  • Storyboard e Auto Layout
  • UIKit com UITableView
  • UserDefaults para armazenar os dados
  • Extensions para modularização visual
  • Delegates e Protocolos

Competências: Mobile Applications · UIKit · Swift · Xcode · Desenvolvimento iOS · Storyboard · Modularidade · Boas Práticas
