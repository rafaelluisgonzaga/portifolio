ElencoStarWars App

Exploração de personagens com UIKit e API SWAPI

Ver no GitHub

Sobre o Projeto

🌌 ElencoStarWars é um app iOS criado com UIKit que consome dados da API SWAPI para listar e detalhar os personagens da saga Star Wars. A navegação é simples e fluida, com filtros por gênero e uma arquitetura robusta combinando MVC com ViewModel.

💡 Funcionalidades

  • 📃 Listagem paginada de personagens
  • 🎯 Filtro de gênero com Picker
  • 🧑‍🚀 Visualização detalhada dos atributos
  • 🔗 Integração com SWAPI

🛠️ Tecnologias

  • Swift com UIKit
  • Padrões: MVC, ViewModel, Delegates
  • Interface Builder para construção visual
  • API REST da SWAPI.dev

Competências: UIKit · Swift · API Integration · iOS Architecture · Interface Builder · Delegates
