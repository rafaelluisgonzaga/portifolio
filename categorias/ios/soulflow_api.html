Sobre o Projeto

🎧 O SoulFlow | Api Version é um aplicativo iOS que busca e reproduz músicas utilizando a API do iTunes. Com uma interface intuitiva construída em SwiftUI, o projeto destaca-se por sua fluidez, capacidade de resposta e integração eficiente entre front-end e back-end. É uma evolução do App Soul Flow, agora alimentado por dados reais em vez de mockados.

💡 Funcionalidades

  • 🔍 Busca dinâmica de músicas
  • 🎶 Reprodução de áudio com AVPlayer
  • 🌐 Integração total com API iTunes

🛠️ Tecnologias Utilizadas

  • SwiftUI para interface responsiva
  • Combine para estado reativo
  • EnvironmentObject para controle global
  • AVPlayer para reprodução contínua de áudio

Competências: Swift · SwiftUI · Combine · AVKit · API REST · iOS Architecture · App Responsivo · Xcode
